    • Data Types
      • Categorical
        • Definition
          • Data that can take on only a specific set of values representing a set of possible categories.
        • Synonyms
          • Enums
          • Enumerated
          • Factors
          • Nominal
          • Polychotomous
        • Sub-Group
          • Binary
            • Definition
              • A special case of categorical data with just two categories.
                • 0 or 1
                • True or False
            • Synonyms
              • Dichotomous
              • Logical
              • Indicator
              • Boolean
          • Ordinal
            • Definition
              • Categorical data with explicit ordering
            • Synonyms
              • Ordered factor
            • Example
              • Ranking a movie from 1 to 5 stars
      • Numerical
        • Continuous
          • Definition
            • Data that can take on any value in an interval.
          • Synonyms
            • Interval
            • Float
            • Numeric
        • Discrete
          • Definition
            • Data that can take on only integer values, such as counts.
            • Synonyms
              • Integer
              • Count
    • Data Structure
      • Rectangular Data Structure
        • Terminology
          • Dataframe
            • Rectangular data is the basic data structure for statistical and machine learning models
          • Feature
            • A column in a table is called a feature.
            • AKA
              • Attribute
              • Input
              • Predictor
              • Variable
          • Outcome
            • Yes or no result from an inquiry.
            • Features are sometimes used to predict the outcome in an experiment or study.
            • AKA
              • Dependent Variable
              • Response
              • Target
              • Output
          • Records
            • A row in the table.
            • AKA
              • Case
              • Example
              • Instance
              • Observation
              • Pattern
              • Sample
          • Differences in Terminology
            • Sample
              • to a CS this means a single row
              • to a statistician this is a collection of rows
            • Be aware that DS, CS and Statisticians use different terms that may have similar meaning and vice versa. Same terms with different meaning.
        • Dataframes & Indexes
          • Indexes can improve performance
          • Multilevel/hierarchical indexes may improve efficiency in some cases
        • This is the building block for predictive modeling
      • Nonrectangular Data Structure
        • Time Series
          • Successive measurements of the same variable.
        • Spatial Data
          • Used for mapping and location
        • Object Representation
          • The focus of the data is an object (a house) and its spatial coordinates.
        • Field view
          • Focus is on small units of space and the value of a relevant metric.
        • Graph structures
          • Networking optimization
          • Recommender systems
